# @aiter/aiter

Core runtime library for aiter - provides the infrastructure for building AI terminal user interfaces.

## Features

- **UI Components**: React components for terminal interfaces (ChatContainer, Messages, etc.)
- **Trigger System**: Command processing and registry
- **MCP Integration**: Model Context Protocol client management
- **Chat Management**: Session CRUD operations
- **Agent Resolution**: Dynamic agent loading system

## Installation

```bash
bun add @aiter/aiter
```

## Usage

```typescript
import { ChatContainer, initializeMCP, getAgent } from '@aiter/aiter';
```
